# Promptly Frontend

## Environment Setup

Copy `.env.example` to `.env` and update with your backend URL:

```bash
cp .env.example .env
```

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```
